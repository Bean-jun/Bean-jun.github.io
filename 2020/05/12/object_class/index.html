Java 对象和类

  1. 对象

    对象是类的一个实例（对象不是找个女朋友），有状态和行为。例如，一条狗是一个对象，它的状态有：颜色、名字、品种；行为有：摇尾巴、叫、吃等。

  2. 类

    类是一个模板，它描述一类对象的行为和状态。
